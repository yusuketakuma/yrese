## 位置づけ

医療機関等側の第7.0版GL(SEC-001)に対し、本書は**サービス提供事業者(当社)側**の遵守事項を扱う。
両者の分担は「医療機関等は利用者としての管理責任 / 事業者はシステム・サービスの安全管理責任」であり、
契約(SLA・責任分界・データ処理契約)で明文化する(§9.8)。

## 分担早見(医療機関等側 GL との突き合わせ)

| テーマ | 薬局(利用者)側 | 当社(事業者)側 |
|---|---|---|
| 利用者アカウント管理 | 職員の発行・停止申請、退職時報告 | 失効処理の即時反映、オフライン認証TTLでの残存リスク制御(SEC-005) |
| バックアップ | 薬局内媒体の保管(該当時) | Cloud/Edge 両系の自動バックアップ・復元試験 |
| 監査ログ | 定期確認・不審操作の報告 | 改ざん検知可能な形式での記録・保全・提供 |
| BCP | 薬局内の非常時業務手順(LOCAL_ONLY 訓練) | モード設計・復旧支援・訓練材料の提供 |
