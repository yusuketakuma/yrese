## 本台帳の位置づけと限界(実装前に必読)

1. 本台帳の evidence は **点数値・点数表上の規定** に対するものである。**算定要件(いつ算定できるか)の evidence ではない** — 算定要件は留意事項通知(EVD-002、未精読)の精読後に別途発行する。したがって EVIDENCE_ISSUED は「golden test 期待値の作成が開始できる」状態であり、APPROVED_FOR_IMPLEMENTATION ではない(CAL-001 解除手順 step 3〜5 が残る)。
2. 出典PDFはテキスト抽出不可のため**誌面判読(CONFIRMED_VISUAL)**である。golden test 作成時に原本該当ページとの再照合を必須とする。
3. 令和8年6月19日付とされる修正版の有無が未確認。確認の結果差分が判明した場合は、該当 evidence を SUPERSEDED とし SSOT_UPDATE_REQUIRED を発行する。
4. 適用: **施行 令和8年6月1日**(REG-007)。処方日・調剤日が2026年5月31日以前のデータへ本台帳の点数を適用してはならない(当時有効版管理: REG-002 / MST-001)。
