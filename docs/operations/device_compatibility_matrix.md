## 3. 共通要件

- 印刷失敗を印刷成功扱いにしない(v0.1.7 §9.4 禁止)— 印刷キューは「投入済み≠出力済み」を区別し、出力完了証跡(ジョブ完了+必要に応じ現物確認)を残す
- 二重印刷防止: 再出力は理由記録+権限確認+出力履歴に「再出力」明示(帳票SSOT §20 と同期)
- 2次元シンボル読取エラーを手入力補完で隠さない — 読取失敗は読取ログに残し、手入力補正は照合ステップ(紙面照合)を経る
- デバイス接続失敗を薬剤師確認済み扱いにしない
- サポート対象デバイスリストを公開し、対象外条件を曖昧にしない(検証環境で動作確認した機種のみ「サポート対象」と表示)
